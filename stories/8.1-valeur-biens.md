# Story 8.1 - Valeur estimée des biens

## Objectif
Permettre d'indiquer une valeur estimée (en euros) pour chaque objet, et ainsi calculer la valeur totale des biens choisis par chaque membre de la famille.

## Contexte utilisateur
Les parents souhaitent pouvoir estimer la valeur des objets pour :
- Avoir une vue d'ensemble du patrimoine catalogué
- S'assurer d'une répartition équitable entre les enfants
- Identifier les objets de valeur importante

## Fonctionnalités

### 1. Champ valeur sur la fiche objet
- Champ optionnel "Valeur estimée" en euros
- Format : nombre entier ou décimal (ex: 150, 1500.50)
- Affiché sur la page détail de l'objet
- Modifiable par tous les membres de la famille (comme les autres champs)

### 2. Affichage dans la galerie
- Badge discret sur la vignette si une valeur est renseignée
- Format court : "150 €" ou "1,5k €" pour les gros montants

### 3. Statistiques par personne
- Nouvelle section ou page "Répartition"
- Pour chaque membre : liste des objets "Je le veux !" avec valeur totale
- Affichage : Nom + nombre d'objets + valeur totale
- Mise en évidence des écarts importants entre membres

### 4. Statistiques globales (optionnel)
- Valeur totale du patrimoine catalogué
- Nombre d'objets avec valeur renseignée vs total
- Valeur moyenne par objet

## Critères d'acceptation

### Saisie de la valeur
- [ ] Le champ valeur accepte les nombres positifs (0 autorisé = gratuit/sans valeur)
- [ ] Le champ est optionnel (null = valeur inconnue)
- [ ] Format d'affichage avec séparateur de milliers français (1 500 €)
- [ ] Cible tactile minimum 44px pour le champ de saisie

### Affichage galerie
- [ ] Badge valeur visible mais discret sur les vignettes
- [ ] Ne pas afficher de badge si valeur non renseignée

### Calcul répartition
- [ ] Tableau récapitulatif par personne
- [ ] Seuls les objets avec préférence "Je le veux !" sont comptés
- [ ] Les objets sans valeur sont comptés en nombre mais pas en euros
- [ ] Tri par valeur totale décroissante

## Notes techniques
- Nouveau champ `estimatedValue` (decimal/float) dans le schéma Item
- Endpoint pour récupérer les stats de répartition
- Formater les montants avec `Intl.NumberFormat('fr-FR', { style: 'currency', currency: 'EUR' })`
