# Story 3.2 - Ajouter ou modifier la description d'une fiche

## Epic
Epic 3 : Enrichissement collaboratif des fiches

## User Story
**En tant qu'** utilisateur  
**Je veux** ajouter ou modifier la description d'une fiche  
**Afin de** donner des détails sur l'objet (dimensions, état, origine...)

## Contexte
La description permet de préciser des informations utiles : "Table en noyer massif, 6 places, quelques rayures sur le plateau. Achetée chez Roche Bobois en 1995."

## Critères d'Acceptation

- [ ] Sur la page de détail, une zone description est disponible
- [ ] Si vide, afficher "Ajouter une description..." cliquable
- [ ] Si existante, cliquer dessus permet de l'éditer
- [ ] Zone de texte multi-lignes (textarea)
- [ ] Sauvegarde automatique après quelques secondes d'inactivité
- [ ] Pas de limite de caractères (raisonnable)
- [ ] Le texte est affiché formaté (retours à la ligne préservés)

## Éléments Techniques

### API Endpoint
```
PATCH /api/items/:id
Body: { description: "Table en noyer massif..." }
→ Retourne la fiche mise à jour
```

### Base de données
```sql
UPDATE items SET description = $1 WHERE id = $2 RETURNING *;
```

### Affichage avec retours à la ligne
```jsx
// Préserver les retours à la ligne
<p style={{ whiteSpace: 'pre-wrap' }}>
  {description}
</p>

// Ou avec des <br>
{description.split('\n').map((line, i) => (
  <React.Fragment key={i}>
    {line}
    {i < description.split('\n').length - 1 && <br />}
  </React.Fragment>
))}
```
